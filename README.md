**Sistema de Cálculo de Média por Aluno**

Projeto desenvolvido com HTML, CSS e JavaScript puro, com o objetivo de praticar manipulação de DOM, arrays de objetos e lógica de programação.

**Funcionalidades**

✅ Adicionar nome do aluno

✅ Adicionar múltiplas notas para o mesmo aluno

✅ Calcular média automaticamente

✅ Exibir média individual de cada aluno

✅ Limpar todos os dados

✅ Adicionar nota usando botão ou tecla Enter

**Conceitos praticados**

- Manipulação do DOM

- addEventListener

- Eventos de clique e teclado

- Arrays de objetos

- Métodos como:

  - `.find()`
  
  - `.forEach()`
  
- Validação de inputs

- Cálculo de média

- Atualização dinâmica da tela

**Objetivo do Projeto**

Este projeto foi desenvolvido para:

- Consolidar conhecimentos em JavaScript

- Praticar lógica com objetos

- Simular um pequeno sistema real

- Evoluir no aprendizado como desenvolvedor web
